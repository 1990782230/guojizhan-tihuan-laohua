# 最终套图的6个Prompt

程序在印花替换完成后，以同一张 `02_pattern.png` 为输入，并发执行以下6个独立Prompt：

| 文件 | 输出目标 |
|---|---|
| `03_main.txt` | 1张完整电商场景主图 |
| `04_detail_01.txt` | 包身印花与皮革纹理微距 |
| `04_detail_02.txt` | 手柄与连接工艺微距 |
| `04_detail_03.txt` | 肩带与金属连接件微距 |
| `04_detail_04.txt` | 顶部包边与缝线微距 |
| `04_detail_05.txt` | 侧边转角与包边工艺微距 |

每个文件都是一套完整、独立、可编辑的Prompt。程序每次运行都会读取文件的最新内容。
