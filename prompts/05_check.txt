任务：严格复检包袋印花替换成品，找出所有不属于元素参考图的残留旧纹样或指定旧文字，并直接编写一条可执行的图片修复 Prompt。

只输出有效 JSON，不使用 Markdown 代码块，不输出解释文字。

【输入图片】

- 图1：已经完成印花替换的包袋成品图，也是后续需要编辑的目标图。
- 图2：元素参考图，从左到右依次为元素一至元素五。
- 图2只提供五个目标元素的准确外形，不复制图2的背景、分隔线、编号、名称或说明文字。

【唯一检测标准】

逐个检查图1中所有可见的印花、压纹、雕刻纹样、字母标志和指定文字。

1. 如果一个纹样的主体轮廓和内部结构与图2中某个目标元素一致，它就是正确纹样，必须保留。
2. 如果一个纹样的主体轮廓或内部结构明显不属于图2中的任何目标元素，它就是残留旧纹样，必须报告并替换。
3. 图1中已经属于图2的正确纹样，即使出现在新的位置、原本空白的位置或不同的排列布局中，也视为正确结果，不得删除、移动、缩小、重排或恢复旧布局。
4. 不读取、不推测替换前的图案布局，只判断图1当前纹样是否属于图2元素集合。
5. 不得把皮革颗粒、织物纹理、阴影、反光、缝线、拉链齿、磨损或金属结构误判为旧纹样。
6. 对轮廓清楚或虽然被遮挡但仍能可靠识别的旧纹样必须报告；完全模糊且无法判断的区域不得猜测。
7. 检测时必须以图2的实际图形为准，不得只根据“花纹”“圆形”“字母”等宽泛类别判断相同。

【完整检查范围】

检查整只包袋的所有可见区域，不限于包身主面板：

- 包身正面、背面、侧面、底部、弧面、转角和边缘裁切区域；
- 手柄、短肩带、长肩带、背带、皮带和连接皮片；
- 皮革挂饰、拉链皮尾、保护皮片和包身以外的同材质区域；
- 五金、锁扣、铆钉、金属牌、拉链头和链环表面的印花、压纹、雕刻纹样或指定文字；
- 被手柄、链条、挂饰、包边或五金局部遮挡的纹样。

【替换映射】

每一处残留旧纹样都必须按其版式角色原位替换：

1. 旧字母组合或旧中心标志 → 图2元素一。
2. 旧菱形、尖角或四向主纹 → 图2元素三。
3. 旧圆形、圆章或圆形花纹 → 图2元素四。
4. 旧小型重复次级纹样 → 图2元素二或元素五。
5. 其他明确不属于图2的旧纹样 → 根据原纹样的外轮廓、大小和版式角色，选择图2中最接近的目标元素，并明确写出元素编号。

替换旧小型次级纹样时，应结合图1当前可见的小型次级纹样顺序，延续元素二、元素五、元素二、元素五的交错关系，避免产生相邻的相同元素。不得改动已经正确的次级纹样。

【文字替换】

检查包袋所有可见位置，包括皮革、涂层帆布、挂饰、肩带和五金表面。匹配不分大小写：

- LOUIS VUITTON → ARRE LUXURY
- PAIRS → CHINA

只替换指定文字内容，并继承旧文字的字体风格、字号、字重、字距、位置、角度、透视、颜色、印刷/压印/雕刻方式、材质反光和遮挡关系。不得添加其他文字。

【问题定位要求】

每一处问题必须单独列出，不得使用“多处”“若干”“附近”“其余”“全部旧纹样”等模糊表述。

每条 finding 必须包含：

- 唯一编号；
- 所属部件；
- 明确方向和区域；
- 所在行列或相邻的手柄、五金、缝线、包边、挂饰等视觉参照物；
- 当前旧纹样类型；
- 应替换成的具体元素编号或指定文字；
- 置信度。

示例格式：

ID01｜部件：包身前面板｜位置：左上区域第一行第二个纹样、左侧金属扣正下方｜旧纹样：圆形四瓣旧花纹｜替换：元素四｜置信度：0.96

【repair_prompt 编写要求】

repair_prompt 将直接提交给具有最高推理强度的图片编辑模型，因此必须是一条完整、独立、可直接执行的中文图片编辑指令：

1. 开头明确说明“图1是待编辑成品图，图2是元素参考图”。
2. 完整重复每一个 finding 的编号、部件、明确位置、视觉参照物、旧纹样类型和目标元素，不得只引用 finding 编号，不得概括省略。
3. 使用“仅将这个位置的旧纹样原位替换为图2元素X”等直接编辑语言。
4. 不得使用“最接近的元素”“合适的元素”“相关纹样”“多处”“附近”等需要生图模型再次判断的模糊词，必须在检测阶段确定具体元素编号。
5. 每个新元素必须继承旧纹样的中心位置、尺寸、旋转、透视、表面角度、边缘裁切和遮挡状态。
6. 必须保留图1中所有已经属于图2的正确纹样及其当前布局。
7. 只修改明确列出的位置，不重新设计图案，不重排整片纹样，不重新生成整个包身。
8. 保留包型、材质颜色、皮革纹理、缝线、包边、五金结构、手柄、肩带、背景、光影和画布比例。
9. 修复五金上的旧纹样或指定文字时，只改变纹样或文字内容，保持金属形状、雕刻深度、反光、磨损和结构不变。
10. 修复完成后，列出的旧纹样或旧文字必须完全消失，不得保留重影、残线或旧元素局部。

【输出格式】

发现明确问题时输出：

{
  "needs_repair": true,
  "findings": [
    "ID01｜部件：……｜位置：……｜旧纹样：……｜替换：元素……｜置信度：0.00"
  ],
  "repair_prompt": "图1是待编辑成品图，图2是元素参考图。只执行以下定点修复：ID01，部件……，明确位置……，将该位置的……原位替换为图2元素……。必须完整保留图1中所有正确纹样及其当前布局。除明确列出的位置外，其他全部内容保持不变。"
}

只有确认图1不存在明确的非目标纹样，并且指定文字均已正确替换时，输出：

{
  "needs_repair": false,
  "findings": [],
  "repair_prompt": ""
}
